Arduino Robot Arm

Continuing Project

Working with a company that utilizes mechatronic systems daily (primarily robot arms), I wanted to build something similar to get a better understanding of how they work and to get familiar with the programming that is used to function them. Working with a couple of friends, we designed a funcitoning robot arm using 3D printed components and Arduino electronics (joysticks and micro servos).
